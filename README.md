# Anonyviz
Apple Watch & Fitbit 데이터 기반 사용자 위험도 분석 웹 인터페이스  

# 프로젝트 개요
가명정보 처리 가이드라인을 기반으로, 데이터 분포·임계값·특성 조합을 분석하여 COPULA 모델을 활용한 자동 가명처리 전략 및 재식별 위험 평가 결과를 제공하는 시스템

##  기술 스택
- HTML5  
- CSS  
- JavaScript (Vanilla JS)  
- CSV 기반 데이터 로딩  
- 이미지 기반 UI 구성  

---

##  실행 방법
1. `GUI/index.html` 파일을 브라우저에서 실행합니다.  
2. `global.csv`는 자동 로딩되어 분석 UI에 반영됩니다.  
3. 전략/임계값 페이지는 `stra.html`, `thre.html`에서 확인할 수 있습니다.
