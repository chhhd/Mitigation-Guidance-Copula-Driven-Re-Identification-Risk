# Anonyviz
Apple Watch & Fitbit 데이터 기반 사용자 위험도 분석 웹 인터페이스  

# 프로젝트 개요
이 프로젝트는 Apple Watch 및 Fitbit 센서 데이터를 기반으로 사용자의 건강 관련 위험도를 분석하고,  
이를 웹 기반 GUI 형태로 시각화한 데모 애플리케이션입니다.


---

## 📊 사용된 데이터셋
Apple Watch and Fitbit Data — Kaggle  
https://www.kaggle.com/datasets/aleespinosa/apple-watch-and-fitbit-data

---

## 🖥️ 기술 스택
- HTML5  
- CSS  
- JavaScript (Vanilla JS)  
- CSV 기반 데이터 로딩  
- 이미지 기반 UI 구성  

---

##  실행 방법
1. `GUI/index.html` 파일을 브라우저에서 실행합니다.  
2. `global.csv`는 자동 로딩되어 분석 UI에 반영됩니다.  
3. 전략/임계값 페이지는 `stra.html`, `thre.html`에서 확인할 수 있습니다.
